About the guild

We're a small, laid-back RS3 clan built around one idea: progress is more fun when someone's watching your grind and cheering it on. Whether you're chasing max cape, working through Dungeoneering floors, or just killing time in Prifddinas, there's always someone here doing something similar.

No strict activity quotas, no drama — just a group that shows up for boss nights, helps with quests nobody wants to solo, and tracks each other's XP gains mostly so we have something to rib each other about.

How this works. A GitHub Actions workflow fetches everyone's RS3 hiscores directly from Jagex every 2 hours and appends a snapshot to data/history.json in this repo. This page reads that file directly, so there's no CORS proxy in the loop at all — that fragility from the earlier prototype is gone. Weekly XP is the difference between the most recent snapshot and the snapshot closest to 7 days before it, so figures firm up as more history accumulates.
