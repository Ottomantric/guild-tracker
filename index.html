How this works. A GitHub Actions workflow fetches everyone's RS3 hiscores directly from Jagex every 6 hours and appends a snapshot to data/history.json in this repo. This page reads that file directly, so there's no CORS proxy in the loop at all — that fragility from the earlier prototype is gone. Weekly XP is the difference between the most recent snapshot and the snapshot closest to 7 days before it, so figures firm up as more history accumulates.
